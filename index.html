STATISTICS

We have started an online survey in order to gather information regarding how people are currently dealing with the problem of not having the right items when needed. We wanted to have an overview about the situation in order to know how to design Kitley to fit best most people's needs.

First we asked them when did they need the items. This will tell us how often would the basic user use the app. The more recent, the better for us.



Then we would like to know how urgent that need was. If it was a very important situation, and a high need, then the users will be more likely to use our pinging system.



Lastly, we asked a general queestion, whether they got it or not in the time frame. Regardless if it wes hard or not, there were still users that did not get the item, which again is good news for us.
